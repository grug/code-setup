## Diagram Drawing

<https://excalidraw.com/>

I like Excalidraw because:

- No logins required.
- It's free.
- It has a no BS interface.
- Easy to export to SVG/PNG or to get a direct link to the diagram.
- Live sharing at the touch of a button (also doesn't require logins).
